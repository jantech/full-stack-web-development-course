
# Code Editor : Visual Studio Code

# Recommended VS Code Extensions for React JS

| Extension                              | Purpose                                   |
| -------------------------------------- | ----------------------------------------- |
| ES7+ React/Redux/React-Native Snippets | React shortcuts & component snippets      |
| Prettier - Code Formatter              | Auto format code                          |
| ESLint                                 | Find code errors & warnings               |
| Highlight Matching Tag                 | Highlight opening & closing tags          |
| Tailwind CSS IntelliSense              | Tailwind class suggestions                |
| Path Intellisense                      | Auto complete import paths                |
| Auto Rename Tag                        | Rename paired HTML/JSX tags automatically |
| Auto Close Tag                         | Automatically close tags                  |
| Material Icon Theme                    | Better file & folder icons                |
| Thunder Client                         | API testing inside VS Code                |

---
